Slide JS

Basic slide instance

        
  new Slide();

This will set up all of the needed slide events and animations. {{slide_id}} can be used inside of injections and will inject the key of the slide.

Slide Events

        
  new Slide({
    onRendered: function() {
      // this will run when the slide is added to the DOM
    },
    onReady: function() {
      // this will run when the slide comes into view
    }
  });

The onRendered and onReady functions are called with the Slide as the context. So calling this will reference the Slide object.
